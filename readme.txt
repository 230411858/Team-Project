How to run and use this code:

- Make sure you have the apache webserver installed
- Make sure php is installed
- Make sure composer is installed
- Make sure npm is installed
- Make sure you have a mySQL server running
- Unzip the archive file into your home directory
- In your home directory create a symlink to the the GamerHub/public folder named public_html
- Change directory into the GamerHub folder and run the following commands:
- npm install
- npm run build
- composer install
- Then set the .env variables to the values corresponding to your mySQL server
- Copy the contents of the cs2team42_db.sql file into your database
- You should now be able to access the website
